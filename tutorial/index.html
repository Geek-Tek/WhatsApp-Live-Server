Welcome to my WhatsApp Live Chrome extension

The new way of communicating, better than instant messaging


Here's how it works



It simply does everything automatically.

When you open a chat, it searches if the user has the extension.

If he does, the live chat starts and you will be able to comunicate in real time!

You'll see a small green dot under the user picture if the live chat is started


Note: this extension works only with WhatsApp Web



What about...


If you already had WhatsApp Web opened during the download?

Just reload the page, and the application will work perfectly!


If something doesn't work as it should?

You can help me making my extension better. Just press the extensions icon in the up right corner of your browser and then the WhatsApp Live icon. In the menu click the report a bug button and choose the best way for reporting your bug.


Can I use the extension with WhatsApp Desktop?

The short answer is no. Sadly, the extension works changing the HTML page, and I can't interact with the desktop app. If you want to use this extension you need to be on web.whatsapp.com


How can I help with the extension?

If you click the extensions icon in the up right corner of your browser and then the WhatsApp Live icon, you can press the "Help the project" button and help me out in several ways. Here's how: suggest the extension to your friends, send an idea, make a donation or help me on GitHub
